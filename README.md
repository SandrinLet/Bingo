🎡 Roleta de Bingo
Este projeto foi desenvolvido como material de apoio para aulas de Javascript que eu estava ministrando na escola Mentes Criativas.

🎯 Objetivos de Aprendizado:
Manipulação de arrays (push, includes).

Estruturas de repetição controladas (while).

Interação com o DOM (getElementById, createElement).

🚀 Como funciona:
O sistema sorteia números de 1 a 75 sem repetição, vinculando-os às letras tradicionais do Bingo (B-I-N-G-O) e mantendo um histórico visual.
